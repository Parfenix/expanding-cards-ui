# Expanding Cards

A smooth, interactive UI component where panels expand on click with dynamic gradient background transitions.

## ✨ Features

* Smooth expanding card animation
* Dynamic background gradient transitions
* Keyboard navigation (← →)
* Responsive layout
* Clean and minimal UI

## 🛠 Tech Stack

* HTML5
* CSS3 (Flexbox, transitions)
* Vanilla JavaScript
